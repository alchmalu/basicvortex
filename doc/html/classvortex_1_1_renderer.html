Detailed Description

Abstract class that must render a scenegraph. User willing to implement their own rendering system must derive this class and implement all virtual methods.

This class has a reference to the scene manager. there is, for the moment, no way to have scene manager interact with renderers. see todo page for more information

think and implement a way to have bidirectional interaction between scene manager and renderers.

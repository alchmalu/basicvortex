Detailed Description

Material representation class. Is responsible for storing Texture manner of use along with it.

Member Typedef Documentation

typedef std::map<TextureSemantic, Texture *> vortex::Material::TextureMap
protected

Associated Textures with this material.

The map associate a semantic (diffuse, ambient, normal, ... to the texture. Available semantics are AMBIENT, DIFFUSE, SPECULAR, EMISSIVE, HEIGHT, NORMALS, SHININESS, OPACITY, DISPLACEMENT, LIGHTMAP, REFLECTION, UNKNOWN

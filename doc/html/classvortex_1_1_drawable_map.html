Detailed Description

template<typename Key, typename Child>
class vortex::DrawableMap< Key, Child >

Generic map (based on std::map) with a draw method. Drawing a map results in drawing all its elements. A map may be drawn with or without parameters. ACTUAL LIMITATION : when rendering loops are nested, only the outer loop may have parameters
